Visual Studio Code(Vs Code)

What is Vs Code

Visual Studio Code (VS Code) is a lightweight, powerful source code editor that's available for Windows, macOS, and Linux

Why we use vs Code

Visual Studio Code (VS Code) is a popular code editor that developers use for a variety of reasons, including:

  • Speed:VS Code is fast and lightweight, making it ideal for day-to-day use
  • Features: VS Code includes many features to help you write code, including:
    • Syntax highlighting, bracket-matching, and auto-indentation
    • IntelliSense code completion
    • An interactive debugger
    • Support for Git
    • Built-in tools for cleaning up and organizing code
  • Customization: You can customize VS Code to look and work the way you like, by changing its settings and adding new tools from the Visual Studio Marketplaceï»¿
  • Platform Support:VS Code works on different types of computers and can be easily installed on any platform
  • Language Support:VS Code supports hundreds of languages, including built-in support for Node.js, TypeScript, and JavaScript
  • Integration:VS Code integrates with build and scripting tools to perform common tasks

Features of Vs Code

Visual Studio Code (VS Code) is a code editor with many features, including:

  • Debugging: An interactive debugger that allows users to step through source code, inspect variables, and more
  • Version Control: Built-in support for Git, which allows users to work with source control without leaving the editor
  • Code Completion:IntelliSense code completion helps users write code faster by predicting what they'll type next
  • Command Pallete:Users can access commands by typing a keyword in the Command Palette, which can be opened with the key combination Ctrl+Shift+P
  • Keyboard Shortcuts: Intuitive keyboard shortcuts allow users to navigate their code with ease

Extensions of VS Code

1. Live Server Launches a local development server with live reload for static files.

Emmets of VS Code:

Here's a table of some commonly used Emmet symbols in VS Code and their purposes:

S.No Emmet Symbol Purpose
1. > Child element creation
2. + Sibling element creation
3. ^ Move up to the parent element
4. * Generate multiple elements
5. {} Add custom text inside an element
6. [] Add attributes to an element
7. # Add an id attribute
8. . Add a class attribute
9. : Add pseudo-classes or pseudo-elements
10. ! Generate a boilerplate HTML document
